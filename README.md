# Global Air Pollution Analysis

This project performs Exploratory Data Analysis (EDA) on global air pollution data provided by the World Health Organization.

## Objective
To analyze pollution patterns across countries, cities, and regions using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Top polluted countries
- Top polluted cities
- Pollution trend over years
- Regional pollution comparison
- Correlation analysis of pollutants

## Key Insights
- Afghanistan shows the highest average PM2.5 pollution levels.
- Cities such as Bamenda and Kabul show extremely high pollution.
- Western Pacific Region has the highest regional pollution levels.

## Dataset Source
World Health Organization (WHO) Air Quality Database
